# Welcome to the docs!

## Input Format for non-flag use:
firstNumber(Space)Operator(Space)secondNumber
eg.: -1507 x 75757

## Input with Flags(does not work with the desktop launcher!):
gocalc --o Operator --f firstNumber --s secondNumber
eg.: gocalc --o x --f -1507 --s 75757

## Funktions:
+ = Addition
- = Subtraction
x = Multi.
* = Multi.
/ = Division
^ = Power
v = Roots; "2 v n" for Squareroot of n; works not only with Squareroots; all floats are supported
tan & tanh
sin & sinh
cos & cosh
log
% eg.: 25 % 80 = 20
